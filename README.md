# 项目初始化

```bash
git clone https://github.com/dong9205/web_terminal.git
cd web_terminal
go mod tidy
```

# 目录结构说明

* cmd/web_terminal/web_terminal.go: 项目入口文件, WebSocket Server 就在这里实现
* ui: 前端页面目录
* pkg/terminal: WebSocket Terminal 具体的业务实现
* pkg/k8s: k8s 相关功能封装与验证
